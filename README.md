# 割り勘メモ

シンプルな日本語の割り勘アプリです。会計金額の合計と人数を入れるだけで均等割りを確認でき、必要ならオプションでメンバーごとの名前と支払い金額まで細かく入力できます。HTML / CSS / JavaScript だけで作っているため、GitHub Pages にそのまま公開できます。

## アプリ概要

- 純粋な静的ファイルだけで動作
- スマホで見やすいレイアウト
- 会計金額と人数だけで均等割りを即時計算
- 割り切れないときは整数円で自然に調整
- オプションでメンバー名を入力可能
- 人数に応じて `メンバー1` `メンバー2` のように自動生成
- 各メンバーの支払い金額を入力すると残額がリアルタイム更新
- 未入力メンバー向けの残額均等割り目安を表示
- 精算結果を `A → B : 1200円` 形式で表示
- `localStorage` によるデータ保存と復元
- 全データ削除機能

## GitHub Pages での公開方法

1. この4ファイルを GitHub リポジトリのルートに配置します。
2. GitHub に push します。
3. リポジトリの `Settings` を開きます。
4. `Pages` を開きます。
5. `Build and deployment` の `Source` で `Deploy from a branch` を選びます。
6. ブランチに `main`、フォルダに `/ (root)` を選んで保存します。
7. 数分待つと公開URLが発行されます。

## ローカルでの確認方法

ビルドは不要です。`index.html` をブラウザで直接開くだけで確認できます。

## ファイル構成

- `index.html`: 画面の構造
- `style.css`: レイアウトと見た目
- `app.js`: 状態管理、保存、均等割り計算、精算計算
- `README.md`: 説明書

## 使い方

1. 一番上で会計金額の合計と人数を選びます。
2. すぐ下に整数円で調整済みの均等割り結果が表示されます。
3. 必要なら `オプション` を開きます。
4. 各メンバーの名前を任意で入力します。未入力でも `メンバー1` のように使えます。
5. 各メンバーの支払い金額を入力すると、入力済み金額、残額、未入力メンバー向け目安が更新されます。
6. 合計と支払い金額が一致すると、精算結果が表示されます。
